A cosa serve Bootstrap?

Bootstrap è un framework front-end che facilita la creazione di siti web responsive e mobile-first. Fornisce componenti predefiniti e classi CSS per layout, pulsanti, form e molto altro.

Cosa devi importare per creare una pagina web con Bootstrap?

Per utilizzare Bootstrap, devi importare il CSS e JavaScript tramite un CDN. Ecco come:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">
< script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare Bootstrap sul tuo sito?

Un CDN (Content Delivery Network) è una rete di server distribuiti geograficamente che consente di distribuire i file rapidamente. Utilizzando un CDN per Bootstrap:

  • I file vengono caricati più velocemente grazie ai server vicini all'utente.
  • Non devi gestire manualmente gli aggiornamenti dei file.
  • Riduce il carico sul tuo server poiché i file vengono distribuiti dal CDN.

A cosa serve questo attributo del tag meta?

L'attributo content="width=device-width, initial-scale=1" serve a rendere la pagina responsive sui dispositivi mobili. In particolare, imposta la larghezza della pagina alla larghezza del dispositivo e mantiene lo zoom iniziale al 100%.

Che differenza c’è tra Container e Container Fluid?

Container: È un contenitore con una larghezza fissa, centrato sullo schermo.
Container Fluid: Occupa l'intera larghezza della finestra del browser, adattandosi alla dimensione del dispositivo.

Prova a utilizzare il container fluido con questa pagina chiamata BTS_1.1.1.html:

<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Container Fluid Example</title>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">
</head>
<body>
    <div class="container-fluid">
        <h1>Esempio di Container Fluid</h1>
        <p>Questo container si estende per tutta la larghezza dello schermo.</p>
    </div>
</script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
